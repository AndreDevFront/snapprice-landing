# 🎯 SnapPrice — Landing Page

> Descubra o valor real de qualquer item em segundos com IA

Landing page oficial do SnapPrice com demo interativo.

🔗 **Live:** https://andredevfront.github.io/snapprice-landing/

## Stack

- HTML5 + CSS3 puro (sem frameworks)
- Design System com tokens CSS (light/dark mode)
- Demo interativo simulando o app nativo

## Deploy

Este repositório usa **GitHub Pages** para deploy automático.
Qualquer push na branch `main` atualiza o site em ~30 segundos.

## Desenvolvimento local

```bash
git clone https://github.com/AndreDevFront/snapprice-landing.git
cd snapprice-landing
open index.html
```
